# Practica04 - Unity 2D

## Descripción
Proyecto de plataforma 2D con movimiento avanzado, animaciones y sonido.

## Contenido
- Movimiento horizontal con aceleración y desaceleración
- Salto con coyote time y jump buffer
- Animaciones: Idle, Walk, Jump, Fall
- Sonidos: salto, pasos e impacto

## Cómo ejecutar
1. Abrir la carpeta Practica04 en Unity.
2. Abrir la escena principal.
3. Presionar Play.

## Controles
- A / D o Flechas: mover
- Espacio: saltar

## Notas
El jugador usa Animator con parámetros:
- Caminando
- Saltando
- Cayendo
